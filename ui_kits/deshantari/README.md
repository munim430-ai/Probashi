# Deshantari UI Kit

**Product:** Deshantari mobile app  
**Surface:** Mobile (375px wide, iOS/Android)  
**Stack:** React + Babel inline JSX

## Screens
1. **Home** — Dashboard with quick actions, balance, recent transfers
2. **Send Money** — Step-by-step remittance flow
3. **Jobs** — Job listings with filters
4. **Profile** — User profile, documents, settings

## Components
- `Header.jsx` — Top app bar with logo, notifications, avatar
- `BottomNav.jsx` — 5-tab fixed bottom navigation
- `HomeScreen.jsx` — Dashboard screen
- `SendMoneyScreen.jsx` — Remittance flow (3 steps)
- `JobsScreen.jsx` — Job listings + detail view
- `ProfileScreen.jsx` — Profile + documents

## Usage
Open `index.html` — presents the app as a phone-framed interactive prototype.
Click through screens via bottom nav and action buttons.
